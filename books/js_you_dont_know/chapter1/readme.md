何为作用域 链？

执行上下文
函数是一等对象
每一个函数执行时，会先在自己创建的AO对象上找对应属性值
AO 函数的管理者
若找不到则往父函数AO上找，
再找不到，再上一层的AO，
直到大boss window(全局作用域)
AO链 就是Javascript中的作用域链

AO上如果有与函数同名的属性，则会被此函数覆盖

编译阶段 AO生成
比喻于prototype 原型链

LHS和RHS 都是出现在引擎对变量进行查询的时候
左查询，右查询发生了多少次？

=

LHS：变量赋值或写入内存
想象为将文本文件保存到硬盘中。
RHS：变量查找或从内容中读取
想象为从硬盘中打开文本文件

LHS RHS 特性
都会在所有作用域中查询
严格模式下，找不到所需的变量，引擎会ReferenceError异常
非严格模式下，LHR会自动创建一个全局变量
